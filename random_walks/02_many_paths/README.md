# 02 — Many Paths

One walk is a story. A thousand walks is a *pattern*. We run 500 walks at once, plot them as an overlapping fan, animate them growing together, and look at the distribution of where they end up. The std of the final positions turns out to be suspiciously close to √(number of steps) — foreshadowing the next notebook.
